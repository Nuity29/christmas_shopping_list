# christmas_shopping_list
Shopping list exercise in Christmas version using JavaScript/DOM.

1) Think about your loved ones.

2) Write the type of gift and for whom it should be in the "enter items" field.

3) Insert items by clicking on the "ENTER" button or pressing the "enter" key on your keyboard.

4) If you're already bought the gift, you can line-through the item.

5) If you would like to delete any items from the list, click.

Thanks for reading and merry Christmas!
